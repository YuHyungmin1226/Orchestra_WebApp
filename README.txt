=======================================
청소년 오케스트라 출석 관리 시스템
=======================================

이 프로그램은 청소년 오케스트라의 출석을 관리하기 위한 경량 웹 기반 애플리케이션입니다.
인터넷 연결 없이 사용자의 컴퓨터에서 직접 실행됩니다.

---
**실행 방법**
---

1. (최초 1회만) 필요한 라이브러리 설치:
   터미널(명령 프롬프트)을 열고 이 폴더로 이동한 뒤, 아래 명령어를 입력하세요.
   pip install -r requirements.txt
   (컴퓨터에 따라 'pip' 대신 'pip3'을 사용해야 할 수 있습니다.)

2. 프로그램 실행:
   이 폴더에 있는 `run.py` 파일을 실행하세요. (예: `python run.py` 또는 `python3 run.py`)

3. 자동 실행:
   프로그램이 실행되면 자동으로 웹 브라우저에 관리 페이지가 열립니다.

4. 프로그램 종료:
   프로그램을 종료하려면, `run.py`를 실행했던 터미널(까만 창)을 닫으세요. (Ctrl + C 키를 눌러도 됩니다.)


---
**!!! 중요: 데이터 백업 안내 !!!**
---

모든 데이터(단원 정보, 출석 기록 등)는 이 폴더 안의 `data` 폴더에 CSV 파일 형태로 저장됩니다.
USB 분실이나 컴퓨터 고장에 대비하여, **주기적으로 `data` 폴더 전체를 다른 곳에 복사하여 백업**해주세요.
이 프로그램은 자동 백업 기능이 없습니다.


---
**폴더 구조**
---

- `run.py`: 프로그램을 실행하는 파일입니다.
- `data/`: 모든 데이터가 저장되는 폴더입니다. **(중요! 백업 필요)**
- `web_files/`: 프로그램의 로직과 웹 페이지 파일들이 들어있습니다. (수정 불필요)
- `requirements.txt`: 실행에 필요한 라이브러리 목록입니다.
